# Plan 2026 - Mihaela Fitness

## Visie
2025 was een jaar van ontdekken en vernieuwing. 2026 wordt het jaar van **resultaat en consistentie**.

## Doelstelling
Centraal overzicht waar we oude data (2025) kunnen zien, maar vooral een strak plan voor 2026 met focus op:
- Consistentie in training
- Consistentie in voeding  
- Consistentie in waterinname
- Persoonlijke begeleiding per klant

---

## Fase 1: Overzichtspagina (NU) ✅

### Functionaliteit
- **2025 Overzicht sectie**
  - Klanten met resterende sessies van vorige periode
  - Totaal aantal openstaande sessies
  - Statistieken 2025 (trainings, voedingsplannen, etc.)
  
- **2026 Plan sectie**
  - Totaal aantal actieve klanten in 2026
  - Overzicht klanten per status (actief, inactief, nieuw)
  - Quick stats: trainingsessies deze maand, voedingsplannen actief
  - Knoppen om nieuwe features te bouwen

### UI Components
- Stats cards (vergelijkbaar met andere admin pagina's)
- Tabel/overzicht van klanten met resterende 2025 sessies
- Tabel/overzicht van 2026 actieve klanten
- Action buttons voor toekomstige features

---

## Fase 2: Persoonlijke Plannen (TOEKOMST)

### Functionaliteit
Per klant een persoonlijk plan met:
- **Training schema**
  - Wekelijkse trainingsdoelen
  - Specifieke oefeningen
  - Progress tracking
  
- **Voedingsplan**
  - Dagelijks voedingsschema
  - Macros doelen
  - Maaltijden suggesties
  
- **Waterinname doel**
  - Dagelijks doel (bijv. 2L)
  - Tracking per dag

### Database Structuur (idee)
```typescript
// Nieuwe tabellen
PersonalPlan {
  id
  customerId
  year: 2026
  trainingGoal
  nutritionGoal
  waterGoal (liters per dag)
  startDate
  endDate
  status: 'active' | 'completed' | 'paused'
  createdAt
  updatedAt
}
```

## Fase 3: Dagelijkse Taken (TOEKOMST)

### Functionaliteit
Klanten kunnen dagelijks inloggen en taken afvinken:
- ✅ Training gedaan
- ✅ Voeding gevolgd
- ✅ Water doel behaald
- ✅ Andere custom taken per klant

### Database Structuur (idee)
```typescript
DailyTask {
  id
  customerId
  planId (PersonalPlan)
  date
  taskType: 'training' | 'nutrition' | 'water' | 'custom'
  taskName
  completed: boolean
  completedAt
  notes (optioneel)
  createdAt
}
```

### Klant Portal
- Login pagina voor klanten (`/my-plan` of `/client-portal`)
- Dashboard met:
  - Dagelijkse taken checklist
  - Progress overview
  - Weekoverzicht
  - Notities/feedback mogelijkheid

---

## Fase 4: Maandelijkse Check-in (TOEKOMST)

### Functionaliteit
Elke maandag krijgen klanten een check-in formulier met vragen zoals:
- Hoe voel je je deze week?
- Wat ging goed?
- Wat zijn je struggles?
- Energie level (1-10)
- Motivatie level (1-10)
- Gewicht/maten update
- Feedback voor coach

### Database Structuur (idee)
```typescript
WeeklyCheckIn {
  id
  customerId
  planId
  weekStartDate (maandag)
  energyLevel: number (1-10)
  motivationLevel: number (1-10)
  howAreYouFeeling: text
  whatWentWell: text
  struggles: text
  weight?: number
  measurements?: json
  feedbackForCoach?: text
  submittedAt
  createdAt
}
```

### Automatisering
- Elke maandag automatisch check-in formulier versturen (email/notificatie)
- Herinnering als niet ingevuld na 2 dagen
